How were the data processed

The PISA data was obtained here. I only keep the columns of country; the number of cellphone, tv, computer, car and bathroom; and the first plausible values of mathematics, reading and science. The corresponding column names are CNT, ST27Q01, ST27Q02, ST27Q03, ST27Q04, ST27Q05, PV1MATH, PV1READ and PV1SCIE. Then the data is extended with the GDP per capita in 2012 of corresponding countries. GDP data was mainly from world bank. Then the data were processed in file "PISA_exploration.Rmd". It exported several files to folder "generated_data". Some were further processed by script in "reshape_json.py" then the final data for visualization were ready. The world geodata was obtained here with a bit extension to make the country names agree with those in PISA ("clean_geo.py" was used for this purpose.).
